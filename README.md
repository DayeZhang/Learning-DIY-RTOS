# 项目说明
此项目提供课程《自己动手从0到1写嵌入式操作系统》所有配套资料的下载，含源码、参考文档等。

如对该资料有兴趣，可以使用GitHub的打包下载功能。或者使用Git工具、SourceTree等将整个项目克隆到本地。

由于课程存在着一些Bug，所以后期可能会偶尔修改源码。如果想及时获取该项目资料的所有修改，建议使用Git工具克隆整个项目。
